# Tab Cards

一个 Manifest V3 Chrome 插件，用水果卡片风格展示当前窗口的标签页，并支持拖拽管理标签组。

## 功能

- 以卡片方式展示 Tab 标题、链接、图标、Active 状态、收藏状态、重复标签和建议关闭状态
- 展示 Tab 打开时长，安装后从插件首次观察到该 Tab 的时间开始累计
- 优先使用可用的进程内存信息；普通 Chrome 环境下使用页面 JS heap 作为 Tab 内存参考
- 支持搜索、筛选、紧凑视图、关闭单个 Tab、批量关闭建议项
- 支持展示 Chrome 标签组，直接修改组名和颜色
- 支持拖拽 Tab 到已有组、未分组区域，或拖到侧栏投放区新建组
- 支持收藏/取消收藏 HTTP(S) 标签页

## 本地加载

1. 打开 `chrome://extensions/`
2. 开启右上角“开发者模式”
3. 点击“加载已解压的扩展程序”
4. 选择本目录
5. 点击工具栏里的 Tab Cards 图标，会打开独立的 Tab 管理页面

如果已经打开过管理页，再次点击图标会自动聚焦已有页面，不会重复打开很多个管理页。

## 说明

Chrome 稳定版没有稳定开放精确的单 Tab 进程内存 API。插件会在 API 可用时读取进程内存，否则对 HTTP(S) 页面读取 `performance.memory.usedJSHeapSize` 作为内存参考；受限页面会显示为 `--`。
